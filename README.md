# 锲颖供应商平台 Web管理界面

## 项目目录结构
```
 src ────┬──── asserts       # 静态资源文件目录
         │    ├─ css
         │    ├─ image
         │    └─ js
         │
         ├──── components    # 组件目录
         │    ├─ management
         │    └─ supplier
         │
         ├──── config        # 配置文件目录
         │    ├─ bus
         │    ├─ network
         │    ├─ router
         │    └─ vuex
         │
         └──── views         # 视图页面目录
              ├─ management
              └─ supplier
```
